很久没有整过这玩意了，现在都不会配置了。
1：安装根目录下的data 和base 目录一定要指定好。
2：不要自己去建立data文件夹
3：在遇到error 1045 ，被denial的情况。网上的教程 有个错误（如下图2所示）。貌似是最新版本的MySql 在user这个表上做了一些更改。
注意1：如果自定义了 port 口的话，可能会是在 等待连接，你在连接的时候却连接不上！
注意2：Password这个列在最新的user表中已经不存在了！现在已经是authentication_string了！

2: 在使用过程中如果碰到密码过期的问题可以使用下面的命令来解决：
set PASSWORD=PASSWORD(‘new_PASSWORD’)
